# CodeMentor AI 5-Minute Demo Script

## Goal

Show the project as a polished academic full-stack application with authentication, AI-assisted workflows, SQL safety, history, and review rubrics.

## 0:00 to 0:30 - Introduction

Say:

> "This is CodeMentor AI, a full-stack coding assistant built with React, FastAPI, SQLite, and Monaco Editor. It can generate, explain, debug, optimize, review, and document code while keeping each user's history secure."

Click:

- Open the app home page
- Point to the header and workspace layout

## 0:30 to 1:10 - Login

Say:

> "The system uses authentication so each user sees only their own history."

Click:

- Go to the login page
- Enter the demo email and password
- Click `Login`

## 1:10 to 2:10 - Generate Code

Say:

> "First, I can generate code from a natural-language prompt."

Click:

- Choose `Python`
- Select `Generate Code`
- Enter prompt: `Write a function to reverse a string`
- Click `Run AI Action`

Then say:

> "The result appears in the output panel, and I can download the generated code or the full report."

Click:

- Show the generated code panel
- Click `Download Code`
- Click `Download Full Report`

## 2:10 to 3:10 - Review Rubric

Say:

> "The review feature uses a rubric out of 10 and groups issues by severity."

Click:

- Select `Review Code`
- Paste a small code sample into the input editor
- Click `Run AI Action`

Then point out:

- Quality score
- Rubric breakdown
- Severity groups
- Top 3 improvements

Say:

> "The review is advisory. It highlights potential issues and suggests refactoring ideas without changing the program unless needed."

## 3:10 to 4:00 - SQL Safety Mode

Say:

> "When SQL is selected, the app switches to analysis-only mode."

Click:

- Change language to `SQL`

Then point to the notice:

> "SQL Analysis Only - No Query Execution."

Say:

> "This project never connects to a real database or executes user SQL. It only warns about destructive statements, missing filters, and performance concerns."

## 4:00 to 4:30 - History and Security

Say:

> "Every action can be saved to history, and each user's data stays isolated by authentication."

Click:

- Open the history sidebar
- Click a saved history item

Then say:

> "The workspace restores the saved prompt, code, and output for that specific user."

## 4:30 to 5:00 - Closing

Say:

> "In summary, CodeMentor AI combines AI assistance, secure authentication, SQLite history, code review rubrics, and SQL safety into one academic full-stack project."

Finish by highlighting:

- Frontend
- Backend
- Database
- Testing
- Security
